Interviews: Technical¶

Students in Udacity’s Nanodegree programs can submit Dry-Run Interview Projects, which focus on both behavioral and technical aspects of job interviews. These projects teach you techniques on how to prepare for and successfully answer technical and coding practice questions. They also outline common issues, such as “getting stuck” while answering a technical question, and advise you on how to overcome them. The interview project is currently available in the respective Nanodegree programs:

Front-End Web Developer
How would you go about writing a function or analyzing a front-end web application framework?
Data Analyst
What are the different types of data questions you could be asked? How would you tackle a question asking you to construct a query differently from a question about statistical models or probability?
Full Stack Web Developer
What are the components you need to address when answering a question about web security? How would you approach writing code while explaining your reasoning for it?
iOS Developer
How would you address design and development questions using Apple’s iOS Software Development Kit?
Android Developer
How would you design and develop an Android app using the most efficient methods?

These interview courses are also available in Udacity’s Career Advisor programs!

We are currently working to expand these interview projects into other Nanodegree programs. Stay tuned!

Web Practice¶

  • Pramp

    Peer-practice for technical interviews ... for free!

  • interviewing.io

    Practice interviewing with engineers from top companies, anonymously. Get better at algorithmic problems and find a job!

  • InterviewBit

    Learn and practice on almost all coding interview questions asked historically and get referred to the best tech companies!

  • Interview Cake

    This website offers the opportunity to practice coding questions before the actual interview. Paid subscription only.

  • Career Cup

    Look up interview questions from big tech companies, find tips and tricks, and check out tons of other questions here.

  • Udemy Interview Prep

    Comprehensive course which will help you prepare for a technical interview! Paid subscription only.

  • Big O Cheat Sheet

    This webpage gives you a big FREE cheat sheet for general coding and your upcoming technical interview. It includes the best, average and worst case complexities for search and sorting algorithms.

  • List of Programming Interview Questions

    Looking for a list of programming practice interview questions? Look no further - this website gives a long list of questions for you to practice your coding.

  • Project Euler

    Project Euler is a series of 500+ mathematical and computational problems designed to be solved programmatically. It also includes levels, achievements and other community features.

Books¶

If you find the online resources above not sufficient enough for your interview preparation, you might want to consider these books!

Cracking the Coding Interview by Gayle Laakmann McDowell
This best selling book offers 150 programming questions and solutions to help you practice coding and ace your upcoming technical interview questions.
Programming Interviews Exposed: Secrets to Your Next Job by John Mongan, Noah Kindler, Eric Giguère
This is popular guide to programming interviews includes code examples, information on the latest languages, chapters on sorting and design patterns, tips on using LinkedIn, and a downloadable app to help prepare applicants for the interview.
Elements of Programming Interviews: The Insiders’ Guide by Tsung-Hsien Lee and Amit Prakash
This book features a great compilation for programming related problems for interview prep, general refreshers, or simply brain teasers.
