Finding freelance and remote work¶

Truelancer
Truelancer is a Global Community of Trusted Freelancers. Post your projects, find work, and get hired.
Top Coder
This is a competition website that, at its core, is a place to find freelance work.
Freelanship
Another great place to find freelance work.
Free Code Camp
This open source community gives people opportunities to learn to code and then give back by helping non-profit organizations.
Upwork
Upwork is an awesome tool for finding freelance jobs. International students in particular may find this helpful because Upwork handles payment through its own system.
Remote OK
While this resource is not technically about freelance, this is a great resource for finding remote work.
We Work Remotely
This is a great place to find and list jobs that aren’t restricted by commutes or a particular geographic.
