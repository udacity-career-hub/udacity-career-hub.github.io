High Quality Data Science Coursework¶

Bitbootcamp Resources
Helpful list of Resources for Machine Learning, Statistics and Computer Science.
Clare Corthell’s Open Source Data Science Masters
The open-source curriculum for learning Data Science. Foundational in both theory and technologies, the OSDSM breaks down the core competencies necessary to make data useful.
Andrew Ng’s Coursera Machine Learning Class
A broad introduction to machine learning, datamining, and statistical pattern recognition.
Jeff Leek’s Coursera Data Science Class Exploratory Data Analysis Class
Essential exploratory techniques for summarizing data.
Chris Fonnesbeck’s Advanced Statistical Computing Course
Markov Chain Monte Carlo (MCMC), Metropolis-Hastings, Gibbs sampling, estimation-maximization (EM) algorithms, data augmentation algorithms with applications for model fitting and techniques for dealing with missing data.
